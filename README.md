# Web Chat App

* Backend -> Django
* Frontend -> VueJS
* Database -> Postgres
* Admin db -> pgadmin 4

## Backend

### Starting a new project

```
  docker-compose run backend django-admin.py startproject project .

  sudo chown -R $USER .

  npm install -g @vue/cli

  vue create frontend

  docker-compose run frontend npm install
```

### Just Run

```
  sudo chown -R $USER .
  
  docker-compose up
```
